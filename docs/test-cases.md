# 测试与验收清单

## 已执行自动化验证

- 后端：`.\mvnw.cmd test`
- 前端：`npm run lint`
- 前端：`npm run build`

## 新增权限测试

- 学生不能查看其他学生详情。
- 教师不能查看其他教师授课课程详情。
- 学生不能查看与本人无关的课程详情。
- 教师不能为其他教师的课程录入成绩。
- 学生不能查看其他学生成绩。

## 手工验收建议

### 管理员

- 能正常登录并进入仪表盘。
- 可对院系、专业、班级、学生、教师、课程、成绩完成增删改查。
- 可对用户账号完成新增、编辑、删除、重置密码。
- 删除课程时，若存在成绩记录应得到冲突提示。
- 不能删除自己，也不能把自己的账号禁用。

### 教师

- 登录后只能看到学生信息、课程信息、成绩管理菜单。
- 课程页只能看到本人授课课程，且没有新增/编辑/删除课程按钮。
- 成绩页只能录入或修改本人授课课程的成绩。
- 仪表盘统计应只反映本人相关数据。

### 学生

- 登录后只能看到我的信息、我的成绩菜单。
- 学生页只能看到本人档案。
- 成绩页只能看到本人成绩，不出现新增/编辑/删除按钮。
- 仪表盘统计应只反映本人相关数据。

## 回归关注点

- 分页、搜索、筛选在不同角色下仍可正常工作。
- 登录态过期或 token 缺失时应跳回登录页。
- 前端与后端的角色行为应保持一致，不出现前端暴露按钮但后端拒绝的错位体验。
